# Flask_IIS

Reposity for a youtube tutorial on how to run a flask app in IIS.

0. Clone this repo your project folder

1. run: npm install

2. run: npm run build

3. serve flask app

Youtube Tutorial: https://youtu.be/En9vo7Ognm0
